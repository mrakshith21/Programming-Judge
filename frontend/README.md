# Frontend

The Frontend is a React application that provides a user interface for the Programming Judge. It allows users to view problems, submit solutions, and track their submission status.

## Overview

- **React**: UI Library
- **Tailwind CSS**: Styling
- **Monaco Editor**: Code editor for submissions
- **Axios**: API communication

## Prerequisites

- **Node.js**: 18.x or higher
- **npm**: 9.x or higher

## Installation

1. Navigate to the frontend directory.
2. Install dependencies:
```bash
npm install
```

## Run

1. Start the development server:
```bash
npm start
```
2. The application will be available at `http://localhost:3000`.
